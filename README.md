# Catálogo de Produtos - Spring Boot + PostgreSQL

Este repositório atualmente contém o arquivo compactado do projeto:

- `catalogo.rar`

## Como acessar o código-fonte

1. Extraia o arquivo `catalogo.rar` em uma pasta local.
2. Abra a pasta extraída no seu editor/IDE.
3. Execute os comandos do projeto a partir da pasta onde estiver o `pom.xml`.

## Observação importante

As instruções de execução (Java, Maven, PostgreSQL e `application.properties`) dependem dos arquivos internos do projeto, que estão dentro do `.rar`.  
Se você quiser, posso também revisar e corrigir o README **interno** após a extração.
